# Web App Skeleton

## Purpose

This is the step-7 frontend shell. It intentionally contains only static placeholder
panels and no live API wiring yet.

## Files

- `index.html`
  - single-page console layout
- `styles.css`
  - responsive panel styling
- `app.js`
  - static readiness check for required panels
- `favicon.svg`
  - local icon to avoid asset 404 noise during preview

## Local Preview

From repository root:

- `python3 -m http.server 4173 --directory apps/web`

Then open:

- `http://127.0.0.1:4173`
